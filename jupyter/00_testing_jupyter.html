Notebooks¶

Notes taken from this tutorial: https://www.youtube.com/watch?v=HW29067qVWk

Running "jupyter notebook" starts a server on our local host 8888.

Kernels are basically what programming language you want to use.

In Jupyter, a "kernel" is a computational engine that executes the code contained in a Jupyter notebook. It is responsible for running the code in the notebook's code cells and interpreting the programming language used in the notebook.

Noteboks have two different modes: command mode and edit mode.

Command mode allows us to perform actions like adding and deleting cells. You can access command mode by pressing the Esc key.

The edit mode allows us to alter the cells. You can access edit mode by pressing the Enter key.

In [1]:
print("Hello world!")

Hello world!

Shortcuts:

  • Ctrl + Enter = Run cell
  • Shift + Enter = Run cell and select below in command mode
  • Alt + Enter = Run cell and add new one below in edit mode
  • m to select markdown mode
  • y to select code mode

If we include an exclamation mark before the code, the interpreter will run it as a bash command.

In Jupyter notebooks, "magics" are special commands that allow you to perform various actions that are not part of standard Python syntax. Magics are prefixed with one or two percent signs (% or %%) and are used to control the behavior of the notebook or to interact with the underlying system.

There are two types of magics in Jupyter:

  1. Line Magics: These magics are prefixed with a single percent sign (%) and apply to a single line of code.
  2. Cell Magics: These magics are prefixed with two percent signs (%%) and apply to the entire code cell.
